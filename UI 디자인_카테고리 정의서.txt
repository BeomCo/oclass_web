[product] 제품 리스트
{DB - sql}
 - db/product.sql (
     id : 상품 등록자 아이디 / 
     name : 상품 등록자 이름 / 
     title : 상품 제목 / 
     sub : 상품 부제 /
     content : 상품 소개 내용
     price : 상품 가격 /
     fav : 좋아요 횟수 /
     hit : 상품 조회수 /
     regist_day : 상품 최초 등록일 /
     file_name : 상품 이미지 이름 /
     file_type : 상품 이미지 형식(jpg, jpeg, png, gif 제한 가능) / 
     file_copied : 실제 DB에 저장될 이름 및 형식
     )
